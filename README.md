### 💫 About Me

I am a **Data Analyst** specializing in **Artificial Intelligence and Data Science**. I bridge the gap between technical data layers and real-world impact by building robust data wrangling workflows, cloud warehousing solutions, and predictive algorithms.

*   🔭 **What I Do:** Optimize SQL queries in Google BigQuery, perform Exploratory Data Analysis (EDA), and deploy interactive Streamlit web apps.
*   🧠 **Specialized Area:** Training regression pipelines, evaluating model metrics, and building LLM integrations.
*   ⚡ **Philosophy:** Data is most valuable when it offers a clear, actionable path to solving a problem.
